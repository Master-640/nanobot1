## 八、技术实现细节

### 8.1 Token记录系统

#### **记录格式**
```plaintext
YYYY-MM-DD HH:MM:SS | prompt=X, completion=Y, total=Z
```

#### **数据性质**
- **prompt**: 会话级累计值
- **completion**: 单次调用值
- **total**: prompt + completion
- **计数器重置**: 会话重启时重置

#### **实现位置**
- **文件路径**: `/root/nanobot1/personal/token_usage.txt`
- **记录时机**: 每次LLM调用后自动记录
- **实现代码**: `nanobot/providers/openai_compat_provider.py`

### 8.2 准确计算方法

```python
def calculate_accurate_tokens(start_time, end_time, records):
    """
    准确计算时间段内的token使用
    
    参数:
        start_time: 开始时间
        end_time: 结束时间  
        records: token记录列表
    
    返回:
        total_prompt, total_completion, total_tokens
    """
    # 1. 筛选时间段的记录
    period_records = filter_by_time(records, start_time, end_time)
    
    # 2. 获取时间段前的最后一条记录
    prev_record = get_last_record_before(records, start_time)
    
    # 3. 初始化累计值
    total_prompt = 0
    total_completion = 0
    
    # 4. 处理第一条记录（考虑重置）
    if period_records:
        first = period_records[0]
        if prev_record and is_reset(prev_record, first):
            total_prompt += first.prompt
        elif prev_record:
            total_prompt += (first.prompt - prev_record.prompt)
        else:
            total_prompt += first.prompt
        
        total_completion += first.completion
    
    # 5. 处理后续记录
    for i in range(1, len(period_records)):
        prev = period_records[i-1]
        curr = period_records[i]
        
        if is_reset(prev, curr):
            total_prompt += curr.prompt
        else:
            total_prompt += (curr.prompt - prev.prompt)
        
        total_completion += curr.completion
    
    # 6. 返回结果
    total_tokens = total_prompt + total_completion
    return total_prompt, total_completion, total_tokens
```

### 8.3 重置检测算法

```python
def is_reset(prev_record, curr_record):
    """
    检测是否发生计数器重置
    
    重置条件:
    1. prompt值大幅下降(>30%)
    2. 时间间隔合理(<5分钟)
    3. completion值正常范围
    """
    if prev_record.prompt <= 0:
        return False
    
    drop_ratio = (prev_record.prompt - curr_record.prompt) / prev_record.prompt
    time_diff = (curr_record.timestamp - prev_record.timestamp).total_seconds()
    
    # 重置条件
    if (drop_ratio > 0.3 and           # 下降超过30%
        time_diff < 300 and            # 5分钟内
        curr_record.completion < 10000): # completion正常
        return True
    
    return False
```
